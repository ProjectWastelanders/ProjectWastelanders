Hello Engine

A C++ / OpenGL Game Engine made by video game development students Zhida Chen & Adrià Sellarés. Hello Engine is a fully featured game engine for Windows platform, with Level Editor, Entity Component System, Custom File Format, Scripting system, Undo/Redo, Resource Manager, Serialization system, and many others. This was a university assignment made in 3~ months.

During this 4 month we have used The scrum methodology in order to organize our team and production plan, following the main structure with weekly sprints, daily scrums and finally retrospectives in order to improve.
